# Proto Shop
A minimal game like Swords & Potions, Tiny Shop, Shop Heroes.

Shop management with:
* Pathfinding (for customers)
* Items and inventory
* Recipes and crafting
* UI and Dialogs
* Simple money system

Uses [DOTween](https://dotween.demigiant.com/), [game-icons.net](https://game-icons.net).
